## 重点语法


	表示某一商店的短语中,可以包shop省略
		the butcher's
		the hairdress's

		英文中时态的变化时通过动词体现的

		一般过去时:
			1. 描述过去的事实或状态
			2. 描述过去的动作
		结构:
			# 陈述句
			S + was/were
			S + .V(e)d
			# 否定句
			S + wasn't/weren't
			S + didn't + .v(原型)
			# 疑问句
			Was/Were + S ?
			Did + S + .v ?
			# 回答
			Yes,S + was/were/did
			No,S + wasn't/weren't/didn't
		动词变化
			1. + ed
			2. + d
			3. 辅音 + y,去y为i + ed
			4. 短元音 + 辅音字母,双写辅音字母 + ed

		# 否定疑问句表示双重肯定
		I can swim.
		I can't swim.
		Can't you swim.
